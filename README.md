# Dwarf in a Bubble
Created for Hack Club Campfire game jam!
Theme: Under the Surface

## Description 
A platformer where you play as a dwarf who travels underwater in a bubble as well as above ground. Avoid spikes and projectile-spitting nautiluses and pufferfish which will pop your bubble and make you drown. Drowning = bad.

Available to play on [https://snej55.itch.io/dwarf-in-a-bubble](https://snej55.itch.io/dwarf-in-a-bubble)
